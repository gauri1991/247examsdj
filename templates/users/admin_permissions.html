About Permissions

Permissions control what actions users can perform in the system. They are automatically generated by Django for each model and can be assigned to users directly or through groups.

  • Create (add_) - Allow users to create new objects
  • View (view_) - Allow users to view objects
  • Update (change_) - Allow users to modify existing objects
  • Delete (delete_) - Allow users to delete objects
